✨ 云端版特点

  • 本地 + 云端双存储 - 灵活选择数据存储位置
  • 多 Workspace 管理 - 轻松管理多个门店、仓库
  • 团队协作 - 成员邀请、角色权限管理
  • 跨设备同步 - 数据实时同步，多端访问
  • AI 智能分析 - 基于 AI 的数据洞察和建议
  • 离线可用 - 本地 Workspace 完全离线可用

📋 v1.8.0 更新内容

🎨 UI 升级

  • 界面风格切换：支持 Cupertino、Material、Fluent 三种界面风格自由切换
  • 视觉优化：优化了多个界面的 UI 细节，提升用户体验
  • 账户设置优化：隐私设置一栏置顶，操作更便捷

🔐 隐私与安全

  • 用户邀请设置：新增「允许被邀请加入 Workspace」开关，用户可控制是否接受邀请
  • 管理后台安全增强：标签页关闭/浏览器关闭/无操作 10 分钟自动退出登录

🔧 功能增强

  • 许可证管理：管理后台支持修改许可证的最大用户数和设备数限制

🚀 核心功能

  • Workspace 工作空间管理 - 支持本地和云端工作空间，数据隔离，团队协作
  • 业务数据管理 - 产品、客户、供应商、员工、采购、销售、退货、进账、汇款
  • 数据统计与分析 - 库存统计、销售分析、财务统计、AI 智能分析
  • 数据管理与备份 - 自动备份、数据导入导出、跨 Workspace 迁移
  • 审计日志 - 完整记录所有操作，支持多维度筛选和详细变更记录
  • 团队协作 - 成员邀请、角色权限、实时权限检查
  • 跨平台支持 - iOS、Android、macOS、Windows 全平台覆盖

💡 不需要云端功能？

  • Agrisale 本地版数据完全存储在本地设备
  • 无需网络，完全离线可用
  • 更轻量、更私密
