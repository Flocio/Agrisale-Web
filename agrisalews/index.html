✨ 云端版特点

  • 本地 + 云端双存储 - 灵活选择数据存储位置
  • 多 Workspace 管理 - 轻松管理多个门店、仓库
  • 团队协作 - 成员邀请、角色权限管理
  • 跨设备同步 - 数据实时同步，多端访问
  • AI 智能分析 - 基于 AI 的数据洞察和建议
  • 离线可用 - 本地 Workspace 完全离线可用

📋 v2.0.0 更新内容

🐛 Bug 修复

  • 采购记录权限修复：修复采购记录不能被非产品创建者更改的问题
  • 必须指定员工问题修复：进账、汇款支持不指定经办人创建；编辑时可清除经办人

⚙️ 服务器与配置

  • 默认 HTTPS 地址支持更改：服务器配置中默认 HTTPS 地址（内网穿透）可编辑、可保存，便于迁移域名

🔒 管理后台安全

  • 管理后台安全性强化：生产环境关闭 API 文档；管理员登录限流，防暴力破解

🔔 通知与 UI

  • Workspace 通知：成员操作通知、列表高亮（新建浅绿、更新浅蓝）
  • 版本过低提示 UI：客户端版本低于服务器要求时，提示更新并引导下载
  • 加载 UI 统一、通知 UI 美化、退出动画优化

📊 其他

  • Workspace 数据概览双饼图、创建时间/成员数点击切换
  • 详情窗口创建者 ID 查询、默认模型参数调整

🚀 核心功能

  • Workspace 工作空间管理 - 支持本地和云端工作空间，数据隔离，团队协作
  • 业务数据管理 - 产品、客户、供应商、员工、采购、销售、退货、进账、汇款
  • 数据统计与分析 - 库存统计、销售分析、财务统计、AI 智能分析
  • 数据管理与备份 - 自动备份、数据导入导出、跨 Workspace 迁移
  • 审计日志 - 完整记录所有操作，支持多维度筛选和详细变更记录
  • 团队协作 - 成员邀请、角色权限、实时权限检查
  • 跨平台支持 - iOS、Android、macOS、Windows 全平台覆盖

💡 不需要云端功能？

  • Agrisale 本地版数据完全存储在本地设备
  • 无需网络，完全离线可用
  • 更轻量、更私密
