✨ 云端版特点

  • 本地 + 云端双存储 - 灵活选择数据存储位置
  • 多 Workspace 管理 - 轻松管理多个门店、仓库
  • 团队协作 - 成员邀请、角色权限管理
  • 跨设备同步 - 数据实时同步，多端访问
  • AI 智能分析 - 基于 AI 的数据洞察和建议
  • 离线可用 - 本地 Workspace 完全离线可用

📋 v1.5.2 更新内容

🎨 UI 改进

  • 产品编辑对话框优化：改进同时修改产品名称和供应商时的对话框提示信息，当产品名称也将被修改时，对话框会明确告知用户，添加"产品名称修改将自动同步到所有相关记录"的提示说明
  • SnackBar 提示信息优化：根据实际修改情况显示更精确的成功提示

📋 v1.5.1 更新内容

🐛 重要修复

  • Workspace ID 冲突修复（关键修复）：修复本地和云端 workspace ID 冲突导致的数据混淆问题
  • Workspace 名称重复检查修复：云端 workspace 现在与服务器上所有云端 workspace 比对，全局不能重名

🔧 功能增强

  • Workspace 管理增强：所有 workspace 操作都正确区分本地和云端，确保操作一致性

📋 v1.5.0 更新内容

🐛 重要修复

  • Workspace 数据隔离修复：修复产品更新时跨 workspace 名称冲突检查的 bug
  • 数据完整性修复：修复删除/更新基础信息导致的相关问题

✨ 新功能

  • 库存统计日期筛选：支持按日期范围筛选产品的采购、销售、退货记录
  • 级联操作日志记录：完整记录所有级联操作的详细日志追踪

🎨 UI 改进

  • 日志界面优化：统一级联操作日志的 UI 风格
  • 对话框鲁棒性增强：所有编辑对话框增加关联实体缺失检测

🚀 核心功能

  • Workspace 工作空间管理 - 支持本地和云端工作空间，数据隔离，团队协作
  • 业务数据管理 - 产品、客户、供应商、员工、采购、销售、退货、进账、汇款
  • 数据统计与分析 - 库存统计、销售分析、财务统计、AI 智能分析
  • 数据管理与备份 - 自动备份、数据导入导出、跨 Workspace 迁移
  • 审计日志 - 完整记录所有操作，支持多维度筛选和详细变更记录
  • 团队协作 - 成员邀请、角色权限、实时权限检查
  • 跨平台支持 - iOS、Android、macOS、Windows 全平台覆盖
