📱 iOS 无线安装（OTA）

无法连电脑时，可在 iPhone 上直接用 Safari 安装，无需数据线。

  • 需使用 Ad Hoc 签名的 IPA，且该设备的 UDID 已加入描述文件。
  • 请用 Safari 打开下方链接（其他浏览器可能无法触发安装）。
  • 若当前在电脑上浏览，可将链接发到手机（邮件、备忘录、微信等），再在 iPhone 的 Safari 中打开。

✨ 云端版特点

  • 本地 + 云端双存储 - 灵活选择数据存储位置
  • 多 Workspace 管理 - 轻松管理多个门店、仓库
  • 团队协作 - 成员邀请、角色权限管理
  • 跨设备同步 - 数据实时同步，多端访问
  • AI 智能分析 - 基于 AI 的数据洞察和建议
  • 离线可用 - 本地 Workspace 完全离线可用

📋 v2.4.0 更新内容

  • 全新的系统通知功能：官方可通过管理后台向用户推送通知，客户端弹窗展示，支持 Markdown、图片、外部链接和应用内跳转
  • 功能页面与数据对比：新增独立功能页面，支持任意两侧数据源（Workspace 或 JSON 文件）的数据对比
  • 筛选与搜索增强：采购记录支持按进货/退货筛选；5 个业务表和 4 个实体均支持按创建者 ID 筛选和 ID 搜索
  • 实体排序功能：产品、客户、供应商、员工支持按拼音、创建时间、更新时间、序号排序
  • 添加记录时搜索实体：采购/销售/退货/进账/汇款添加记录时支持搜索选择产品/客户/供应商
  • 日志功能增强：支持级联操作筛选日志，修复键盘非正常弹出 bug
  • AI 数据分析升级：数据分析助手功能和体验优化
  • 数据管理与备份界面升级：UI 优化，引入数据对比功能入口
  • 自动登录优化：修复需要手动点击才能触发自动登录的问题
  • 账户界面显示用户 ID
  • 后台管理网页 bug 修复

🚀 核心功能

  • Workspace 工作空间管理 - 支持本地和云端工作空间，数据隔离，团队协作
  • 业务数据管理 - 产品、客户、供应商、员工、采购、销售、退货、进账、汇款
  • 数据统计与分析 - 库存统计、销售分析、财务统计、AI 智能分析
  • 数据管理与备份 - 自动备份、数据导入导出、跨 Workspace 迁移
  • 审计日志 - 完整记录所有操作，支持多维度筛选和详细变更记录
  • 团队协作 - 成员邀请、角色权限、实时权限检查
  • 跨平台支持 - iOS、Android、macOS、Windows 全平台覆盖
