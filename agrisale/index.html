✨ 本地版特点

  • 数据本地存储 - 所有数据完全存储在您的设备上，不上传任何信息
  • 完全离线可用 - 无需网络连接，随时随地使用
  • 隐私安全 - 数据不经过任何服务器，100% 私密
  • 无需注册 - 安装即用，无需创建账户
  • 轻量快速 - 本地运行，响应迅速

📋 v3.2.0 更新内容

🎨 UI 改进

  • 产品编辑对话框优化：改进同时修改产品名称和供应商时的对话框提示信息
  • SnackBar 提示信息优化：根据实际修改情况显示更精确的成功提示

🐛 重要修复

  • 数据完整性修复：修复删除/更新基础信息导致的相关问题
  • 产品名称同步：修复产品名称修改后历史记录未同步的问题
  • 崩溃修复：修复编辑已删除/已改名产品的记录时应用崩溃的问题

✨ 新功能

  • 库存统计日期筛选：支持按日期范围筛选产品的采购、销售、退货记录
  • 级联操作日志记录：完整记录所有级联操作的详细日志
  • 搜索功能增强：支持搜索"未知客户"、"未知供应商"等关键词

🚀 核心功能

  • 业务数据管理 - 产品、客户、供应商、员工、采购、销售、退货、进账、汇款
  • 数据统计与分析 - 库存统计、销售分析、财务统计
  • 数据管理与备份 - 自动备份、数据导入导出
  • 审计日志 - 完整记录所有操作，支持多维度筛选和详细变更记录
  • 跨平台支持 - iOS、Android、macOS、Windows 全平台覆盖
