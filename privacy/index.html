1. Information Collection

1.1 Locally Stored Data

Business data in the App (including but not limited to products, customers, suppliers, purchases, sales, returns, payments, and remittances) is stored locally on your device by default.

Locally stored data includes:

  • User account information (username, password)
  • Product inventory information
  • Customer and supplier information
  • Transaction records (purchases, sales, returns, payments, remittances)
  • Report data
  • App settings and preferences

1.2 Cloud Stored Data (AgrisaleWS Only)

If you use the Cloud Workspace feature in AgrisaleWS, the following data will be synchronized to cloud servers:

  • All business data in Cloud Workspaces
  • Team member information and permission settings
  • Collaboration-related operation logs
ğŸ’¡ Note: You can freely choose between Local Workspaces and Cloud Workspaces. Data in Local Workspaces is never uploaded to the cloud.

1.3 Network Request Data

The App may make network requests in the following situations:

a) Version Update Check

  • Purpose: Check if a new version of the App is available
  • Data: Only the current app version number is sent
  • Servers: App official website, GitHub

b) AI Data Analysis Feature (Optional)

  • Purpose: Provide intelligent data analysis services
  • Data: Query content and related business data you actively send to AI
  • Note: This feature requires you to configure your own API key. Data will be sent to the AI service provider you choose (such as OpenAI, DeepSeek, etc.)
  • Warning: Please do not include sensitive personal identity information in AI queries

c) Cloud Sync (AgrisaleWS Only)

  • Purpose: Enable multi-device data sync and team collaboration
  • Data: Business data in Cloud Workspaces
  • Servers: AgrisaleWS cloud servers

2. Use of Information

The information we collect is only used for:

  • Providing core functionality services of the App
  • Checking for app updates
  • Improving app performance and user experience
  • Enabling cloud data sync and team collaboration (AgrisaleWS only)

3. Information Sharing

We do not sell, trade, or otherwise transfer your personal information or business data to third parties, except:

  • With your explicit consent
  • As required by law
  • When you choose to use third-party AI services, relevant query data will be sent to that service provider

4. Data Security

We take the following measures to protect your data:

  • Local Workspace data is stored entirely on your device without network transmission
  • User passwords are encrypted and stored locally
  • Cloud data transmission uses HTTPS encryption
  • Data backup and recovery features are supported

5. Data Control

You have complete control over your data:

  • View: You can view all data stored in the App at any time
  • Export: You can export data in CSV or PDF format
  • Backup: You can create data backups at any time
  • Delete: You can delete any data or uninstall the App to delete all local data
  • Cloud Data Deletion: You can delete Cloud Workspaces and all their data through the App

6. Children's Privacy

This App is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

7. Changes to Privacy Policy

We may update this Privacy Policy from time to time. Updated policies will be posted within the App or on our official website. We recommend that you review this policy periodically.
